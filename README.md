## Installtion

```sh
$ git clone git@github.com:ganmacs/marmot.git
$ sbt run
```

## Runing tests

```
$ sbt test
```

## Environment

* `scala  2.11.7`
* `java 1.7.0_79`
* `sbt 0.13.9`
